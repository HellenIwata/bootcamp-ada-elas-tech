# Bootcamp Elas + Tech

O *Elas+ Tech* é uma iniciativa exclusiva para mulheres e está dentre os projetos pilotos selecionados no programa Caixa: Desafio Mulheres em STEM, uma iniciativa do Fundo Socioambiental CAIXA em parceria com a Artemisia. A formação é 100% gratuita e oferece 1.500 vagas em trilhas de Back-end, Front-end ou Dados, com conteúdos digitais ao vivo e gravados, além de mentorias para impulsionar o desenvolvimento profissional.
